乡旅汇设计方案（MVP版本）

项目目标
打造一个简单易用的乡村旅游体验平台，连接农户与游客。支持：
- 农户发布乡村体验活动
- 农特产品展示
- 基础信息展示与预约
- 简单数据统计

角色与权限设计

| 角色       | 权限描述                                                     |
|------------|--------------------------------------------------------------|
| 👤 游客     | 浏览活动、在线预约、留言评价                                 |
| 🧑‍🌾 农户   | 发布体验活动、上架农产品、查看预约                           |
| 🛠 管理员   | 管理用户、内容管理                                           |

核心功能模块

1. 乡村体验模块
1.1 体验活动发布（农户）
- 标题、活动简介、时间、地点、人数上限
- 上传活动图片
- 可选活动标签：#农事体验 #民宿 #采摘

1.2 活动预约（游客）
- 填写基本信息（姓名+手机号）
- 选择预约日期
- 查看预约状态

1.3 活动展示
- 首页活动列表
- 简单分类筛选
- 活动状态：可预约/已满

2. 农特产品模块
2.1 商品管理（农户）
- 商品信息维护
- 价格设置
- 上传商品图片

2.2 商品展示（游客）
- 商品浏览
- 留言咨询
- 获取联系方式

3. 互动功能
- 活动评价
- 图片分享
- 留言板

技术架构

1. 前端
- Vue.js 构建
- 响应式布局
- 页面：首页、活动详情、商品列表、个人中心

2. 后端
- Python + Django
- RESTful API
- 基础用户系统
- 文件上传

3. 数据库
- MySQL
- 核心表：用户、活动、商品、预约、评价

页面结构
[首页]
└── 活动列表
    └── 活动详情
        ├── 介绍
        ├── 预约
        ├── 评价
        
[农户中心]
├── 活动管理
├── 商品管理
├── 预约查看

[游客中心]
├── 我的预约
├── 我的评价

核心亮点 ✅

| 亮点 | 说明 |
|------|------|
| 🏡 便捷发布 | 农户轻松发布体验活动 |
| 📱 简单预约 | 游客在线预约无障碍 |
| 🌾 特产展示 | 帮助农户展示农特产品 |
